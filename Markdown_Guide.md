# Markdown Guide (based upon https://simplemde.com/markdown-guide)

## Emphasis


`**bold**` **bold**

`*italics*` *italics*

## Headers

```
# Big header
## Medium header
### Small header
#### Tiny header
```

## Lists

```
* Generic list item
* Generic list item
* Generic list item

1. Numbered list item
2. Numbered list item
3. Numbered list item
```

## Links

```
[Text to display](http://www.example.com)
```

## Images

```
![](http://www.example.com/image.jpg)
```
